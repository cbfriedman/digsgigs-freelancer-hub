# Freelancer Job Posting: QA Testing for Marketplace Launch

## Project Title
**Comprehensive QA Testing for Marketplace Web Application - Pre-Launch**

## Project Description

We're seeking an experienced QA Engineer/Tester to perform comprehensive testing on our marketplace web application before production launch. This is a Vite/React/TypeScript application with Supabase backend integration.

### What We've Already Prepared
- **46+ Visual Regression Tests** (Playwright) covering 6 viewports and 5 browsers
- **40+ E2E Functional Tests** (Playwright) for critical user flows
- **Comprehensive QA Deliverables Document** (`QA_DELIVERABLES.md`) outlining all test requirements
- Staging environment with full test data access

### Your Responsibilities

#### 1. Execute All Test Suites (5-7 days)
- Run and document results from existing Playwright test suites
- Execute manual functional testing per `QA_DELIVERABLES.md`
- Cross-browser testing (Chrome, Firefox, Safari, Edge)
- Responsive design testing (Desktop, Tablet, Mobile)
- Performance testing (Lighthouse reports)
- Security audit (authentication, RLS policies, data protection)
- Accessibility testing (WCAG 2.1 Level AA compliance)

#### 2. Integration Testing (2-3 days)
- Stripe payment flows (test mode)
- Email notifications (Resend/Supabase)
- SMS notifications
- Database operations
- Edge functions
- File uploads

#### 3. Bug Documentation & Reporting (Ongoing)
- File detailed bug reports with:
  - Severity classification (Critical, High, Medium, Low)
  - Steps to reproduce
  - Expected vs actual behavior
  - Screenshots/videos
  - Browser/device information
- Daily status updates via agreed communication channel

#### 4. User Acceptance Testing (3-5 days)
- Coordinate with 5+ real users for end-to-end testing
- Collect and document user feedback
- Identify usability issues

### Required Deliverables

11. **UAT Results** (.xlsx survey results + usability issues)
12. **Final Sign-Off Document** (.pdf)
